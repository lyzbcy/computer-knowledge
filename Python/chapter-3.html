第三章 · 数据结构（list/tuple/set/dict）—— 四种容器与推导式

list 可变 ｜ tuple 不可变 ｜ set 去重无序 ｜ dict 键值对 ｜ del 删除 ｜ 列表推导式 ｜ 含 … 道互动关卡

01 · 学习目标

学完这一章你要能做到

  • 分清四种容器：list（可变有序）、tuple（不可变）、set（去重无序）、dict（键值对），各自用什么符号、可不可变。
  • 操作 list：会 append（追加）、insert（插入）、pop（弹出）、sort（排序），知道这些方法原地修改返回 None。
  • 知道 tuple 不可变：括号创建，单元素要加逗号 ('hi',)，支持解包 x,y = t。
  • 用 set 去重：花括号或 set() 创建，自动去重，支持集合运算（并 | 交 &）。
  • 用 dict 存键值对：键唯一不可变，用 del 删键，in 查键，items() 遍历。
  • 写列表推导式：[x for x in ...]，能用任何可迭代对象，不只是列表。
阅读路径：背景包 → 结构地图 → 核心判断 → 关键概念 → 做关卡题 → 生成报告。
本章定位

这是 Python 学习地图的第三章。第一章你学了列表初步，第二章你学了控制流和函数。本章把"数据容器"讲透——四种核心数据结构 + 推导式，它们是 Python 处理数据的利器。内容全部基于 Python 3.12 官方教程第 5 章《数据结构》。

原文依据：Python 3.12 官方教程第 5 章《数据结构》——涵盖列表详解（方法）、del 语句、元组、集合、字典、列表推导式。
02 · 零基础背景包

读这章前先知道的 4 件事

背景①什么是"数据结构"

数据结构就是装数据的容器。不同的容器有不同的特性：有的可改、有的不能改、有的自动去重、有的用名字查值。选对容器，代码就简洁高效。Python 内置了四种最常用的：list、tuple、set、dict。

读这章需要它：理解为什么要有四种而不是一种。
背景②可变 vs 不可变

"可变"指创建后还能改内容（list、dict、set）；"不可变"指创建后不能改（tuple、字符串）。可变性决定了能不能用 append、能不能当字典的键、能不能当 set 的元素。

读这章需要它：这是区分四种容器的核心维度之一。
背景③有序 vs 无序

list、tuple 是有序的：元素按放入顺序排列，能用下标访问。set 是无序的：元素没有固定顺序、不能按下标取。dict 在 3.7+ 保持插入顺序，但本质是用键查值。

读这章需要它：理解为什么 set 不能 s[0] 取值。
背景④键值对的概念

dict 不用连续整数当下标，而用"键（key）"查"值（value）"。键可以是字符串、数字等不可变类型。这就像查字典——给词（键）找释义（值），比按下标翻页快得多。

读这章需要它：理解 dict 为什么比 list 更适合"按名查值"。
03 · 这章在回答什么

表面 vs 深层问题

表层：list/tuple/set/dict 怎么创建、怎么增删改查。

深层：Python 为什么要提供四种容器而不是一种？它们各自解决了什么问题？为什么 tuple 不可变而 list 可变？推导式凭什么比 for 循环更简洁？

只记一个问题：面对一份数据，我该用 list、tuple、set 还是 dict？它们各自的创建方式、可变性、增删改查规则是什么？

带着这个问题读「05 核心判断」和「06 关键概念」，它们都在回答它。
04 · 结构地图

四种容器 + 推导式

官方教程第 5 章按"最常用 → 最灵活"的顺序展开：先深入 list（最常用），再讲 del（删除），然后是 tuple（不可变）、set（去重）、dict（键值对），最后是推导式（高效生成）。

1list 列表 ⭐｜可变有序，[] 创建，append/insert/pop/sort 操作。
2tuple 元组｜不可变，() 创建，用于固定不变的序列、函数多返回值。
3set 集合｜去重无序，{} 或 set()，支持并交差运算。
4dict 字典 + 推导式 ⭐｜dict 用键查值；推导式 [x for x in ...] 一行生成新列表。
节点关系：四种容器按"是否可变""是否有序""是否键值对"区分。其中 list 和 dict 是日常用得最多的，tuple 用于不可变场景，set 用于去重。
05 · 核心判断

6 条核心判断（每条带官方原文依据）

事实① list 是可变有序的列表：append 追加、insert 插入、pop 弹出末尾元素。

append(x) 在末尾加元素；insert(i,x) 在位置 i 插入；pop() 弹出末尾元素（可用于栈）。这些都是原地修改。

原文依据：Python 3.12 官方教程第 5 章《列表详解》——「list.append(x)：在列表的末尾添加一个元素……list.insert(i, x)：在给定的位置插入一个元素。」
事实② tuple（元组）是不可变的，由若干逗号分隔的值组成，常用括号创建。

tuple 一旦创建不能修改。空元组 ()；单元素元组要加逗号 ('hello',)（否则只是括号里的值）。支持解包 x, y, z = t。

原文依据：Python 3.12 官方教程第 5 章《元组和序列》——「元组由若干逗号分隔的值组成……元组是不可变的，但元组可以包含可变对象。」
事实③ set 是由不重复元素组成的无序容器，用花括号或 set() 创建，自动去重。

set 用于去重和成员检测。{'apple','orange','apple'} 自动变成不重复的集合。注意：空集合必须用 set()，{} 是空字典！

原文依据：Python 3.12 官方教程第 5 章《集合》——「集合是由不重复元素组成的无序容器。基本用法包括成员检测和消除重复元素。注意：想要创建空集合，你必须使用 set()，而不是 {}。」
事实④ dict（字典）是键值对的集合，键唯一且必须是不可变类型；用 del 删键、in 查键。

dict 以键（不是连续整数）为索引。键必须是不可变类型（字符串/数字/只含不可变项的 tuple），不能用列表做键（列表可变）。键唯一。

原文依据：Python 3.12 官方教程第 5 章《字典》——「字典以关键字为索引……不能用列表做关键字，因为列表可以用索引、切片或 append() 和 extend() 等方法改变。」
事实⑤ del 语句按索引/键删除元素，也可删除切片或整个变量；不同于有返回值的 pop()。

del a[0] 删列表第 0 项；del tel['sape'] 删字典键；del a[:] 清空列表；del a 删整个变量（之后引用 a 报错）。del 没有返回值，和 pop() 不同。

原文依据：Python 3.12 官方教程第 5 章《del 语句》——「del 语句可以从列表中按索引而不是值来删除一个项……这不同于有返回值的 pop() 方法。」
判断⑥ 列表推导式 [x for x in ...] 从序列简便地创建列表，可遍历任何可迭代对象（不只是列表）。

推导式在 for 后跟一个表达式，再有零或多个 for/if。[x**2 for x in range(5)] 一行生成 [0,1,4,9,16]。它能遍历任何可迭代对象（range、字符串、tuple 等）。

原文依据：Python 3.12 官方教程第 5 章《列表推导式》——「列表推导式提供了从序列创建列表的简便途径。每个列表推导式都在 for 之后跟一个表达式，然后有零或多个 for 或 if 子句。」
06 · 关键概念拆解

6 个关键概念

① list 列表（可变有序）

定义：方括号 [] 创建的有序元素集合，可变。常用方法：append/insert/pop/sort。

fruits = ['orange', 'apple', 'pear', 'banana']
fruits.append('grape')     # 末尾追加
fruits.insert(0, 'kiwi')   # 在位置0插入
fruits.sort()              # 排序（原地，返回 None）
fruits.pop()               # 弹出末尾元素
fruits.count('apple')      # 计数
注意：insert/remove/sort 等仅修改列表的方法都返回 None（这是 Python 可变数据结构的设计原则），不要写 lst = lst.sort()。

② tuple 元组（不可变）

定义：逗号分隔的值，常加括号。不可变——创建后不能增删改元素。

t = 12345, 54321, 'hello!'   # 元组打包
t[0]                         # 12345
x, y, z = t                  # 序列解包

empty = ()                   # 空元组
singleton = 'hello',         # 单元素必须带逗号！
len(singleton)               # 1
不要误解成：单元素写 ('hello') 是字符串不是元组，必须 ('hello',) 带逗号。

③ set 集合（去重无序）

定义：花括号或 set() 创建，元素不重复、无序。支持集合运算。

basket = {'apple', 'orange', 'apple', 'pear'}
print(basket)        # 去重：{'apple','orange','pear'}
'orange' in basket   # True（快速成员检测）

a = set('abracadabra')
b = set('alacazam')
a - b    # 差：在a不在b
a | b    # 并：在a或b
a & b    # 交：既在a也在b
a ^ b    # 对称差：不同时存在
空集合用 set()，{} 是空字典！集合推导式：{x for x in 'abracadabra' if x not in 'abc'}。

④ dict 字典（键值对）

定义：{键: 值}，键唯一且不可变。用键存取值，del 删键，in 查键。

tel = {'jack': 4098, 'sape': 4139}
tel['guido'] = 4127     # 添加/修改
tel['jack']             # 4098（按键取值）
del tel['sape']         # 删除键
'guido' in tel          # True（查键是否存在）

# 遍历键值对
for k, v in tel.items():
    print(k, v)
字典推导式：{x: x**2 for x in (2,4,6)} 得 {2:4, 4:16, 6:36}。

⑤ del 语句

定义：按索引/键/切片删除，或删整个变量。无返回值（区别于 pop）。

a = [-1, 1, 66.25, 333, 333, 1234.5]
del a[0]        # [1, 66.25, 333, 333, 1234.5]
del a[2:4]      # [1, 66.25, 1234.5]
del a[:]        # []  （清空列表）
del a           # 删除变量，再引用 a 会报错
区别 pop：pop() 删除并返回被删元素；del 只删除、不返回值。

⑥ 列表推导式

定义：[表达式 for 变量 in 可迭代对象 if 条件]，一行生成新列表。能遍历任何可迭代对象。

# 组合两个列表中不相等的元素
[(x, y) for x in [1,2,3] for y in [3,1,4] if x != y]
# [(1,3),(1,4),(2,3),(2,1),(2,4),(3,1),(3,4)]

# 遍历字符串（可迭代对象不只是列表）
from math import pi
[str(round(pi, i)) for i in range(1, 6)]
# ['3.1', '3.14', '3.142', '3.1416', '3.14159']
不要误解成：推导式只能遍历列表。错！它能遍历任何可迭代对象（range/字符串/tuple/dict 都行）。

09 · 误解与边界

4 个最容易踩的坑

误解①：tuple 创建后能改元素

真相：tuple 不可变，不能增删改元素。要可变得用 list。但 tuple 可以"包含"可变对象（如内含 list）。

原文依据：第5章《元组》——「元组是不可变的，但元组可以包含可变对象。」

误解②：set 是有序的，能 s[0] 取值

真相：set 是无序的，元素没有固定顺序，不能按下标访问 s[0] 会报错。它的价值是去重和成员检测。

误解③：{} 创建的是空集合

真相：{} 创建的是空字典！空集合必须用 set()。这是高频踩坑点。

原文依据：第5章《集合》——「想要创建空集合，你必须使用 set()，而不是 {}。」

误解④：列表推导式只能遍历列表

真相：推导式能遍历任何可迭代对象——range、字符串、tuple、dict、文件等都可以。

边界：本章覆盖 list/tuple/set/dict/del/推导式。未深入：列表当队列效率低（用 collections.deque）、嵌套推导式、zip/enumerate 循环技巧、序列的字典序比较——这些在官方教程同章但属进阶。

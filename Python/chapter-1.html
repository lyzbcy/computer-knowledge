01 · 学习目标

学完这一章你要能做到

  • 说清 Python 是什么：一门解释型、简单易用、但功能强大的编程语言，老田说"如需支持找他"。
  • 区分解释器的两种模式：交互模式（>>>，边敲边看结果）和脚本模式（写进 .py 文件运行）。
  • 用 Python 当计算器：知道 //（取整）、%（取余）、**（乘方）的区别。
  • 操作字符串和列表：索引（含负数）、切片（左闭右开）、知道字符串不可变而列表可变。
  • 能指出至少 2 个 Python 特点（如"缩进是语法不是装饰""不用分号结尾"）。
阅读路径：背景包 → 结构地图 → 核心判断 → 关键概念 → 做关卡题 → 生成报告。

02 · 零基础背景包

读这章前先知道的 4 件事

背景①什么是编程语言

编程语言是你跟电脑沟通的"指令集"。你用它能听懂的语法写下步骤，电脑按步骤执行。Python 是众多编程语言中语法最接近自然语言的一种，所以入门容易。

读这章需要它：否则你不知道 Python 在"干什么"。
背景②解释型 vs 编译型

C/Java 是编译型：代码要先整个编译成机器码才能跑，改一行得重新编译。Python 是解释型：代码逐行解释执行，可以边写边测，省去编译链接的时间。这是 Python 适合快速开发的原因。

读这章需要它：理解为什么 Python 有"交互模式"（边敲边出结果）。
背景③Python 的定位：胶水语言 + 易学

Python 被称为"胶水语言"——能轻松把不同模块（甚至 C 写的）粘在一起用。它语法简单、内置高级数据类型（列表、字典），能用比 C 更短的语句实现同样目标。适合脚本、数据处理、AI、后端。

读这章需要它：知道老田为什么让你学它——适用面广。
背景④缩进在 Python 里是语法

大多数语言用 { } 表示代码块。Python 用缩进（行首空格）。缩进错了，程序直接报错——它不是风格选择，是语法规则。这是 Python 最显眼的特征。

读这章需要它：否则你写的第一段多行代码就会因缩进报错。
03 · 这章在回答什么

表面 vs 深层问题

表层：Python 怎么入门、基础语法怎么写。

深层：Python 凭什么被称为"易学又强大"？它的交互模式、数据类型设计、缩进语法，分别解决了什么问题？

只记一个问题：打开 Python 解释器后，我能用数字、字符串、列表做哪些基本操作？这些操作的规则是什么？

带着这个问题读「05 核心判断」和「06 关键概念」，它们都在回答它。
04 · 结构地图

Python 入门三层递进

官方教程的入门部分是一条从"能干什么"到"怎么用"的递进线：先给动机（开胃菜），再讲工具（解释器），最后练手感（数字/字符串/列表）。

1开胃菜｜Python 能干什么、为什么用它——建立动机。
2解释器 ⭐｜两种模式：交互（>>>）和脚本（.py 文件）。这是你和 Python 对话的入口。
3数据类型初体验｜数字（int/float）、字符串（不可变）、列表（可变）—— 三种最基础的数据容器。
节点关系：开胃菜给动机 → 解释器给工具 → 数据类型练手感。三者缺一不可，但解释器是入口，没有它你跑不了任何代码。
05 · 核心判断

6 条核心判断（每条带官方原文依据）

事实① Python 是一门解释型语言，可以省去编译和链接的时间，可作为交互式语言边写边测。

这是 Python 适合快速开发的根本原因。你打开解释器就能敲一行看一行结果，不用先编译。

原文依据：Python 3.12 官方教程《开胃菜》——「Python 是一门解释型语言，可以省去编译和链接的时间。因此，它可以作为交互式语言使用，写程序时边写边测试，非常方便。」
事实② 解释器有主提示符 >>> 和从属提示符 ...，分别表示单行输入和多行构造的续行。

交互模式下，>>> 是你输入命令的地方；当你写多行结构（如 if/while），换行后会出现 ... 提示你继续输入属于这个结构的行。

原文依据：Python 3.12 官方教程《解释器》——「主提示符：>>>；从属提示符：...。输入多行构造时需要从属提示符。」
事实③ 整数除法 / 总是返回浮点数；要取整用 //，取余用 %。

这是 Python 和一些语言不同的地方。8 / 5 得 1.6（不是 1）。要丢弃小数部分用 //（17 // 3 得 5），要余数用 %（17 % 3 得 2）。

原文依据：Python 3.12 官方教程《非正式介绍》——「注意整数的除法（/）总是返回一个浮点数。如要只取整数结果……可以使用 // 运算符；要计算余数，可以使用 %。」
事实④ 字符串是不可变的，不能通过索引修改某个字符；列表是可变的。

word[0] = 'J'（word 是字符串）会报 TypeError。但 cubes[3] = 64（cubes 是列表）可以。这是两种核心数据类型的关键区别。

原文依据：Python 3.12 官方教程《非正式介绍》——「字符串不能被修改——字符串是不可变的」「与不可变的字符串不同，列表是可变类型，其内容可以被改变」。
事实⑤ 切片 [开始:结束] 是左闭右开：包含开始位置，不包含结束位置。

word[0:2] 取位置 0 和 1 的字符，不取位置 2。这个规则使得 s[:i] + s[i:] 总是等于 s，非常优雅。

原文依据：Python 3.12 官方教程《非正式介绍》——「切片的开始总是被包括在结果中，而结束不被包括。」
判断⑥ Python 用缩进表示语句分组，缩进是语法不是装饰。

while/if 的循环体或条件体，靠行首缩进界定，不是靠 { }。缩进错了程序直接报错。这是 Python 最具辨识度的设计。

原文依据：Python 3.12 官方教程《非正式介绍》——「循环的主体是有缩进的。Python 用缩进而不是括号来表示语句的分组。」
06 · 关键概念拆解

6 个关键概念

① 解释器与交互模式（REPL）

定义：Python 解释器是你和 Python 对话的程序。交互模式下，你敲一行它执行一行并立即显示结果，提示符是 >>>。

$ python3.12
>>> 2 + 2
4
>>> print("hello")
hello
退出用 quit() 或 Ctrl-D（Unix）/ Ctrl-Z 回车（Windows）。

② 数字类型 int / float

定义：整数（int）和浮点数（float）。Python 解释器可当计算器用。

四种除法相关运算符
>>> 17 / 3    # 经典除法，返回浮点数
5.666666666666667
>>> 17 // 3   # 取整除法，丢弃小数
5
>>> 17 % 3    # 取余
2
>>> 5 ** 2   # 乘方（5的平方）
25
不要误解成：/ 和 // 一样。/ 总是返回浮点数，// 才取整。

③ 字符串（不可变）

定义：用单引号 '...' 或双引号 "..." 包裹的文本，两者完全等价。字符串不可变（不能改某个字符）。

>>> 3 * 'un' + 'ium'    # 重复 + 拼接
'unununium'
>>> word = 'Python'
>>> word[0]    # 索引
'P'
>>> word[-1]   # 负数索引从右数
'n'
转义：字符串里要放引号，用 \' 转义，或换另一种引号包裹（"doesn't"）。

④ 列表（可变）

定义：方括号 [] 包裹的有序元素集合，元素类型可不同。列表可变——能增删改。支持索引、切片、拼接。

>>> squares = [1, 4, 9, 16, 25]
>>> squares[0] = 0       # 可变：改值
>>> squares.append(36)   # 追加
>>> len(squares)         # 长度
7
对比字符串：字符串不可变（word[0]='J' 报错），列表可变（lst[0]=9 正常）。这是两者最关键的区别。

⑤ 切片 [start:end]

定义：从序列（字符串/列表）取一段子集。左闭右开：含 start 不含 end。

写法含义示例（word='Python'）
word[0:2]位置0到2（不含2）'Py'
word[2:5]位置2到5（不含5）'tho'
word[-2:]倒数2个到末尾'on'
word[:]从头到尾（浅拷贝）'Python'
原文依据：《非正式介绍》——「切片的开始总是被包括在结果中，而结束不被包括。」

⑥ 缩进 = 语句分组

定义：Python 用行首缩进（通常是 4 个空格）表示一个代码块（循环体/条件体/函数体），不用 { }。

>>> a, b = 0, 1
>>> while a < 10:
...     print(a)      # 这4个空格缩进 = 属于while
...     a, b = b, a+b # 同样缩进 = 同属while体
...
0
1
1
2
3
5
8
不要误解成：缩进只是为了好看。缩进错了程序就报错，它是语法。

09 · 误解与边界

4 个最容易踩的坑

误解①：/ 和 // 一样，都是除法

真相：/ 总是返回浮点数（8/5=1.6）；// 才是取整（17//3=5）。要取整必须用 //。

原文依据：《非正式介绍》——「整数除法（/）总是返回一个浮点数。」

误解②：字符串和列表一样能改

真相：字符串不可变，word[0]='J' 报错。列表可变，lst[0]=9 正常。要"改"字符串只能新建一个。

误解③：切片 [0:2] 包含位置 2

真相：左闭右开——含 0 不含 2。word[0:2] 取位置 0、1。规则：s[:i]+s[i:]=s。

误解④：缩进只是代码风格，多少空格都行

真相：缩进是语法，不是风格。同一代码块必须缩进一致，错了直接报错。Python 用缩进而非 { } 表示分组。

边界：本章只覆盖 Python 入门最基础的部分（解释器 + 数字/字符串/列表）。控制流（if/for/while）、函数定义、字典/集合/元组、异常处理都不在本章——它们在后续章节。
